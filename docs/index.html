Pulsar

Open-source incident management and alerting platform. A modern, self-hosted alternative to Opsgenie.

Get Started Documentation View on GitHub

Everything You Need for Incident Management

A complete platform for managing alerts, incidents, on-call schedules, and team collaboration in real-time.

Alert Management

Create, acknowledge, snooze, assign, and resolve alerts with full lifecycle tracking and priority levels.

Incident Management

Track incidents with responders, timeline, notes, and linked alerts. Full incident lifecycle support.

On-Call Schedules

Flexible scheduling with rotations, participants, and schedule overrides. Know who's on-call anytime.

Escalation Policies

Multi-level escalation rules with configurable targets and delays. Never miss a critical alert.

Team Management

Organize users into teams with role-based member management and access control.

Notifications

Multi-channel notifications via Email, Slack, and Microsoft Teams with user preferences.

Webhooks

Incoming and outgoing webhooks for seamless integration with your existing tools and workflows.

Real-time Updates

WebSocket support for live dashboard updates. See changes instantly without refreshing.

Quick Start

Get Pulsar running in under a minute.

# Clone the repository
git clone https://github.com/nmn3m/pulsar.git && cd pulsar

# Copy environment file
cp .env.example .env

# Start all services
make up

# Run database migrations
make migrate-up

Then open http://localhost:5173 in your browser.
